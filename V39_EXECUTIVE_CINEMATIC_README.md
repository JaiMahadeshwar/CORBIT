# CASEY TITAN X v39 Executive Cinematic

Real changes in this package:

- Premium CASEY wordmark/logo integrated into the frontend brand system.
- Uploaded ElevenLabs voice added as `/frontend/public/audio/casey_voice.mp3`.
- Uploaded music tracks added as local browser audio files, replacing the procedural bleep/synth bed.
- Cinematic demo overlay now uses the ElevenLabs voice, not robotic browser speech.
- Executive typography/button polish added across nav, tabs, command panel, exports, cards and tables.
- Tornado driver chart now falls back to risk-register driver data if the Monte Carlo tornado array is unavailable.
- Upload tab now includes a visible sample Tier 1 contractor estimate challenge flow.
- Hero narrative updated to a truthful Earth-to-orbit project-controls message.
- Build was run successfully with `npm run build`.

Run:

Frontend:
```powershell
cd frontend
npm install
npm run dev
```

Backend:
```powershell
cd backend
pip install -r requirements.txt
python main.py
```

Open: http://localhost:5173

Important: browser audio still requires a user click. Click **Launch Mission Control**, **Run Demo**, or **Music On** to start voice/music.
