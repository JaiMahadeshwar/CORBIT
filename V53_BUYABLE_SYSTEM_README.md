- Run Earth / Space model entry retained.
- Scenario comparison tab added.
- Methodology / assumptions tab added.

## Buyer flow

1. Watch opening film.
2. Run Earth or Space model.
3. Review overview and scenario comparison.
4. Download full pack.
5. Send project / request access from Pricing tab.
