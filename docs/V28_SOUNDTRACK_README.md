# CASEY TITAN v28 Soundtrack Demo Upgrade

This build adds a procedural cinematic soundtrack to the demo theatre.

## What changed
- `Run cinematic demo` starts a browser-generated soundtrack.
- Earth model uses a tighter low-frequency pulse.
- Space model uses a slower, wider atmospheric pulse.
- A Sound on/off control appears in the demo theatre and app header.
- No commercial music files are included, so there are no royalty/licensing issues in the demo build.

## Why procedural audio
Browsers block autoplay audio until a user clicks. The soundtrack starts only after the user clicks a CASEY demo button, which is compliant with browser behaviour and avoids broken autoplay.

## Future upgrade
Replace the procedural score with a licensed MP3 or custom brand score once the product is ready for public marketing.
